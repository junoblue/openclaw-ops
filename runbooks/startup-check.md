# Startup Check Harness

Use this format for startup checks:

A) Active agent id/model/default status
B) Source files loaded (absolute paths)
C) Missing source-of-truth links
D) Recommended next config/doc alignment actions (max 5)

## Required Task Record
- objective
- sources consulted (paths)
- process
- output
- verification

## Guardrail
Ask before modifying OpenClaw config files.
